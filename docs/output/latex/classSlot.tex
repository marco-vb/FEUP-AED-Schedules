\doxysubsection{Detailed Description}
Classe \mbox{\hyperlink{classSlot}{Slot}}. 

Esta classe representa um slot no horário, ou seja uma aula.~\newline
Ela armazena o dia da semana, a hora de inicio, a hora de fim, o tipo, o nome da turma e o nome da unidade curricular.~\newline
