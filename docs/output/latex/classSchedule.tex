\doxysubsection{Detailed Description}
Classe que guarda um horario. 

Esta classe é usada para representar um horario de uma semana.~\newline
Este armazena um conjunto de slots.~\newline
Um slot corresponde a uma aula.~\newline
Pode ser usada para representar um horario de um aluno, de uma turma, de uma unidade curricular, etc.
