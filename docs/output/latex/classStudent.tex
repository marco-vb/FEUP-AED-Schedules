\doxysubsection{Detailed Description}
Classe que representa um estudante. 

Esta classe é usada para representar um aluno.~\newline
Ela armazena o nome, o número e o horario do aluno.~\newline
Além disso armazena um conjunto de pares de strings.~\newline
Cada par de strings corresponde a uma turma e a uma unidade curricular.~\newline
Este conjunto é usado para armazenar as turmass por unidade curricular que o aluno está inscrito.
