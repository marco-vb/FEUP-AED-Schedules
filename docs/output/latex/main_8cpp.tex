Main function. 

Começa a execução do programa.~\newline
Primeiro cria os sets com todos os alunos(students), todos as turmas(classes), todas as unidades curriculares(courses), todos os pares turma e unidade curricular(class\+Course) e todos as aulas (slots).~\newline
Depois lê os arquivos de entrada (read\+All).~\newline
A seguir imprime as opções do menu e processa a opção escolhida.~\newline
