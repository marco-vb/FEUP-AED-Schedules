\doxysubsubsection{\texorpdfstring{getStudentSchedule()}{getStudentSchedule()}}
{\footnotesize\ttfamily \mbox{\hyperlink{classSchedule}{Schedule}} get\+Student\+Schedule (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{classStudent}{Student}} $\ast$}]{student,  }\item[{slot\+Set $\ast$}]{slots }\end{DoxyParamCaption})}



Retorna o horário do estudante. 

A função itera sobre o o set de pares Classes\+Per\+Course do estudante e procura a aula no set de aulas(slots) utilizando o metodo equal\+\_\+range(), uma vez que os estudante terá varias aulas do mesmo par turma unidade curricular.~\newline
A seguir adiciona as aulas encontradas ao schedule.~\newline
Retorna o schedule.~\newline
 Complexidade Temporal\+: O(A$\ast$log(D) + C$\ast$log(N)) onde C é numero de unidades curriculares em que o estudante está instrito, N é o tamanho do set slots, A é o número de aulas do estudante por semana e D é o número maximo de aulas por dia.~\newline

\footnotesize 
\begin{DoxyPre}
Explicaçao:
  Complexidade de cada passo: 1º for loop: C, onde C é numero de unidades curriculares em que o estudante está instrito
                              equal\_range: log(N), onde N é o tamanho do set slots
                              2º for loop: M, onde M é o numero de aulas dessa cadeira por semana
                              addSlot: log(D), onde D é o número máximo de aulas de um dia
  Complexidade: C*(log(N) + M*log(D)) = C*log(N) + A*log(D), onde A é o número de aulas do estudante por semana, logo A = C*M
\end{DoxyPre}
