int main ( )

Main function.

Começa a execução do programa.
Primeiro cria os sets com todos os alunos(students), todas as turmas(classes), todas as unidades curriculares(courses), todos os pares turma e unidade curricular(classCourse) e todas as aulas (slots).
Alem disso cria um vetor de pedidos por processar (requests).
Depois lê os arquivos de entrada (readPendingRequests e readAll).
A seguir imprime as opções do menu e processa a opção escolhida.
